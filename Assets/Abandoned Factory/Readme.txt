First Steps:

First of all activate deferred rendering and linear color space. 

Open (Abandoned Factory\Scenes\DemoScene) to learn how to use the modular prefabs. The modular prefabs work best with the unity snap settings - no vertex snap is required. 
Most of the prefabs use an 8x8m grid.
